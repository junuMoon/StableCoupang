# StableCoupang 리뷰 리워드 서비스 - 제품 기획서

## 핵심 컨셉

"좋은 리뷰는 가치가 있다. 우리는 리뷰의 질을 평가하고, 가치 있는 리뷰어에게 안정적인 디지털 리워드를 지급한다."

## 서비스 가치 제안

- 리뷰 = 광고: 좋은 리뷰야말로 최고의 광고
- 리뷰어 = 마이크로 인플루언서: 신뢰할 수 있는 리뷰어의 의견이 구매 결정에 영향
- 보상 = 스테이블코인: 실제 가치가 있는 KRW 토큰으로 즉시 지급

## 핵심 기능 플로우

```
1. 사용자가 제품 구매 후 리뷰 작성 (텍스트 + 사진)
   ↓
2. AI가 리뷰 품질을 자동 평가 (0-100점)
   - 사진과 텍스트의 일관성
   - 구체적인 사용 후기 포함 여부
   - 정보 전달력
   ↓
3. 평가 점수에 따라 KRW 스테이블코인 보상 즉시 지급
   ↓
4. 리뷰어의 신뢰도(Reputation Score) 누적
   ↓
5. 신뢰도 높은 리뷰어의 리뷰 모아보기 제공
```

## 화면 구성 (Phase 1 MVP)

### 1. 피드 페이지 (/)
- 모든 사용자의 리뷰를 시간순으로 표시
- 각 리뷰 카드 정보:
  - 제품명
  - 리뷰어 이름
  - 리뷰 텍스트
  - 리뷰 이미지 (있는 경우)
  - AI 평가 점수
  - 지급된 리워드 금액
  - 작성 시간
- 무한 스크롤 또는 페이지네이션

### 2. 리뷰 작성 페이지 (/reviews/new)
- 제품 선택 (드롭다운 또는 검색)
- 리뷰 텍스트 입력 (최소 10자)
- 이미지 업로드 (최대 3장)
- 제출 버튼
- AI 평가 진행 상태 표시
- 평가 완료 후 결과 화면:
  - 리뷰 품질 점수 (0-100)
  - 지급된 리워드 금액
  - 현재 KRW 잔액

### 3. 마이페이지 (/dashboard)
- 기존 기능:
  - KRW/XRP 잔액 조회
  - 토큰 충전
  - 거래 내역
- 새로 추가될 기능:
  - 내 신뢰도 점수 (Reputation Score)
  - 내가 작성한 리뷰 목록
    - 제품명, 점수, 리워드, 작성일
  - 리뷰를 통해 받은 총 리워드 금액

## Phase 1 MVP 범위

## 리뷰 평가 기준 (AI 프롬프트)

```
이 리뷰의 사진과 텍스트를 보고 0-100점으로 평가하세요.
평가 기준:
- 사진과 텍스트의 일관성
- 구체적인 사용 경험 포함 여부
- 제품의 장단점 설명 유무
- 정보 전달력

단순히 "좋아요", "별로에요" 같은 짧은 리뷰는 낮은 점수를 주세요.
```

## 리워드 계산 공식

```
reward_amount = (ai_score / 100) * BASE_REWARD
```

| AI 점수 | 리워드 (BASE_REWARD = 1000 KRW 기준) | 평가 |
|---------|--------------------------------------|------|
| 0-20 | 0-200 KRW | 품질 매우 낮음 |
| 21-40 | 210-400 KRW | 짧거나 성의 없음 |
| 41-60 | 410-600 KRW | 보통, 기본 정보만 포함 |
| 61-80 | 610-800 KRW | 좋음, 사진 + 구체적 설명 |
| 81-100 | 810-1000 KRW | 매우 좋음, 신뢰도 상승 |

## 신뢰도 점수 (Reputation Score)

```
user_reputation = average(all_review_scores)
```

- 사용자별 모든 리뷰 점수의 평균
- 마이페이지에서 표시
- 향후 신뢰도 높은 리뷰어 랭킹에 활용

## 사용자 시나리오

### 시나리오 1: 첫 리뷰 작성

1. 사용자가 로그인 후 "리뷰 작성" 페이지(/reviews/new) 접속
2. 리뷰 작성 폼에서:
   - 제품 선택 (드롭다운)
   - 텍스트 입력 (최소 10자)
   - 사진 업로드 (선택, 최대 3장)
3. "제출" 버튼 클릭
4. "AI 평가 중..." 로딩 화면 (2-5초)
5. 평가 완료 화면:
   - "리뷰 품질 점수: 78점"
   - "+ 780 KRW 토큰 리워드 지급 완료"
   - "현재 잔액: 780 KRW"
6. 마이페이지 또는 피드에서 작성한 리뷰 확인 가능

### 시나리오 2: 피드에서 다른 사용자 리뷰 보기

1. 홈(/) 또는 피드 페이지 접속
2. 모든 사용자의 리뷰 카드 확인:
   - 리뷰어 이름
   - 제품명
   - 리뷰 내용 및 이미지
   - AI 평가 점수
   - 받은 리워드 금액
3. 스크롤하며 다른 리뷰 탐색

### 시나리오 3: 마이페이지에서 리뷰 관리

1. 대시보드(/dashboard) 접속
2. 새 섹션: "내 리뷰"
   - 작성한 리뷰 목록 (제품명, 점수, 리워드)
   - 평균 신뢰도 점수 표시
   - 총 리워드 누적 금액
3. "리워드 내역" 섹션
   - 날짜, 제품, 점수, 지급 금액
4. 기존 잔액 조회 및 토큰 충전 기능 유지

## 기술 스택

### AI 평가
- Google Gemini API
- 멀티모달: 이미지 + 텍스트 동시 분석
- 프롬프트는 간단하게 시작, 추후 정교화

### 이미지 저장
- 로컬 파일시스템: `packages/web/public/uploads/reviews/`
- Multer 라이브러리로 업로드 처리
- DB에는 파일 경로만 저장

### 데이터베이스
- PostgreSQL + Prisma ORM
- 새 테이블: Product, Review, RewardHistory

### 블록체인
- 기존 KRW 토큰 발행/송금 시스템 활용
- `/api/token/issue` 엔드포인트 재사용

## 성공 지표 (해커톤 데모용)

- 리뷰 작성 → AI 평가 → 리워드 지급까지 전 과정이 10초 이내 완료
- AI 평가 점수가 리뷰 품질을 반영 (데모에서 여러 샘플 보여주기)
- 실제 KRW 토큰 잔액이 증가하는 것을 시각적으로 확인 가능
- 피드에서 다양한 리뷰를 보며 커뮤니티 느낌 전달

## 향후 확장 가능성 (멘토 미팅용)

- 신뢰도 높은 리뷰어의 추천 제품 리스트
- 리뷰 기반 커뮤니티 형성
- 판매자가 리뷰어에게 직접 리워드 제공 (마케팅)
- 리뷰 NFT화 (신뢰 자산으로서의 리뷰)
- SNS 공유 시 추가 리워드
- 탑 리뷰어 랭킹 및 배지 시스템
